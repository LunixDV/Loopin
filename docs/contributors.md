## How to add yourself to this list

If you have made a contribution to the codebase (such as fixing a bug, adding documentation, or implementing a new feature), please add yourself to the table by following these steps:

1. Create a pull request containing your code changes.
2. Edit this file (`docs/contributors.md`) and append your name, GitHub username, and details of your contribution.
3. Submit the edits as part of your pull request!
